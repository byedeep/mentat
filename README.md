# API Peek

Minimal Next.js scaffold for pasting an API URL, previewing JSON, and generating request snippets.

## Development

```bash
npm install
npm run dev
```

## Structure

- `app/[...path]/page.tsx` - catch-all result page placeholder
- `app/api/proxy/route.ts` - plain JSON proxy endpoint placeholder
- `app/api/snippet/route.ts` - snippet route placeholder
- `components/` - UI placeholders
- `lib/` - backend helper placeholders
- `proxy.ts` - optional request hook placeholder for current Next.js versions
